# Internship Task 3 – College Event Feedback Analysis

This repository contains Task 3 completed as part of the 
Data Science & Data Analytics Internship with FUTURE INTERNS.

## Project Overview
The objective of this task is to analyze student feedback collected
through Google Forms and extract meaningful insights using Python.

## Dataset
- Feedback responses collected from students
- Data includes ratings, experience level, and textual feedback

## Tools & Technologies Used
- Python
- Google Colab
- Pandas
- Matplotlib
- Seaborn
- TextBlob (Sentiment Analysis)

## Analysis Performed
- Overall rating distribution
- Experience level analysis
- Sentiment analysis of student feedback
- Sentiment distribution and comparison with ratings
- Visualization of insights using charts

## Key Insights
- Majority of students provided high ratings for the event
- Neutral sentiment responses were the highest
- Positive and negative sentiments were nearly equal
- Overall feedback indicates good student satisfaction
